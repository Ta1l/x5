### ПЕРЕД РАЗВЕРТЫВАНИЕМ (5-10 минут)

1. **Проверить DNS:**
   ```bash
   nslookup slotworker.ru
   # Должно показать: 62.217.182.74
   ```

2. **Если DNS не настроена:**
   - Перейти в регистратор домена
   - Добавить A запись: `slotworker.ru → 62.217.182.74`
   - Добавить вторую запись: `www.slotworker.ru → 62.217.182.74`
   - Дождаться распространения (5-24 часа)

3. **Проверить SSH доступ:**
   ```bash
   ssh root@62.217.182.74
   # Пароль: *9w1Z*!R7WxH
   ```

### РАЗВЕРТЫВАНИЕ (на сервере, 10 минут)

**Одна команда:**

- ✅ Установит Node.js 20

- ✅ Проверит все компоненты

### ПОСЛЕ РАЗВЕРТЫВАНИЯ (5 минут)
