## 🐛 ЕСЛИ ЧТО-НИБУДЬ НЕ РАБОТАЕТ

### Быстрая диагностика:

```bash
# На сервере выполните:
bash /var/www/x5/server-diagnostic.sh

# Получите детальный отчет всех проблем с рекомендациями решения
```

### Типичные проблемы:

**1. "DNS не разрешается"**
```bash
# Проверить DNS
nslookup slotworker.ru

# Если не работает - DNS еще не распространилась
# Дождитесь 5-24 часа после создания записи в регистраторе
```

**2. "SSL certificate problem"**
```bash
# Переполучить сертификат
sudo certbot --nginx -d slotworker.ru -d www.slotworker.ru --force-renewal
sudo systemctl reload nginx
```

**3. "502 Bad Gateway"**
```bash
# Node.js не запущен
sudo systemctl restart courier-app
sleep 2
sudo systemctl status courier-app
```

**4. "Connection refused"**
```bash
# Порты не открыты - проверить firewall
sudo ss -tlnp | grep -E ":80|:443"
# Если нет - проверить ISP firewall или VPS панель
```
