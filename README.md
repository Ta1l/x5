# Проект "Работа курьером-партнером"

Этот проект представляет собой веб-сайт для приема заявок от курьеров-партнеров. Сайт собирает данные о кандидатах и сохраняет их в локальную базу данных SQLite.

## Структура проекта
- `index.html` — Главная страница с формой заявки.
- `server.js` — Сервер на Node.js, обрабатывающий POST-запросы и сохраняющий данные в БД.
- `courier_applications.db` — Файл базы данных SQLite, в котором хранятся заявки.
- `package.json` — Файл зависимостей проекта.

## API сервера
Сервер запускается на порту 3000 и предоставляет следующие эндпоинты:
- `POST /api/courier-application` — Принимает данные формы и сохраняет их в БД.
- `GET /api/applications` — Возвращает список всех заявок (для администратора).
- `GET /` — Отдает главную страницу `index.html`.

## Зависимости
Проект зависит от следующих npm-пакетов:
- `express`
- `sqlite3`
- `cors`
- `body-parser`
